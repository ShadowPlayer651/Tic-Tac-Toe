🎮 Tic‑Tac‑Toe
A modern, lightweight, and sleek browser‑based Tic‑Tac‑Toe game.

✨ Features
⚡ Fast and lightweight — no frameworks

🎨 Modern, clean UI

📱 Fully responsive

🎯 Smooth gameplay with win detection

🔁 Reset button for quick rematches

🕹️ How to Play
Player X starts

Click any empty tile to place your mark

First to get 3 in a row wins

If the board fills with no winner → it’s a draw
